<h2>1️⃣ What is the difference between var, let, and const?</h2>
<p>var: It is an old version to declare a variables. No need in modern Javascript. var is function scoped or global scoped. If declare inside the function, it is a local to that function. If outside then it is global.</p>
<p>let: On the other hand, let is the modern version to declare a variables. which is follow the block scoped. Mostly we used in count, accumulator. let is a blocked scoped. used in {}.</p>
<p>const: Where else using const we can't reassign the variables. Use const unless we need to reassign. const is a blocked scoped. used in {}. </p>

<h2>3️⃣ What is the difference between map(), filter(), and forEach()?</h2>
<p>map(): It is create a new array of the same length with transformed elements. We use it when we need to transformed data.</p>
<p>filter():On the other hand, It is create a new array of the same length or shorter length with elements that pass through a conditions. We use it when we need a subset of an array.</p>
<p>forEach(): We use it like a traditional for loop, that execute a function for each elements. We use it when we want to do something with each element of array. Here we don't create a new array.</p>
